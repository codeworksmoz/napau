## 📋 Sobre o Projeto

Site institucional e e-commerce para a **Napau Design & Arte**, especializada em topos de bolo personalizados para aniversários, casamentos, baptizados e eventos especiais.

### Funcionalidades

- 🛍️ **Carrinho de compras** com localStorage
- 📱 **Design responsivo** (mobile first)
- 🔍 **Filtro por categorias** de produtos
- 🖼️ **Galeria de imagens** com zoom
- 💬 **Integração com WhatsApp** para encomendas
- 🎓 **Página de curso** com informações sob consulta
- 📞 **Formulário de contacto** via WhatsApp

## 🚀 Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| HTML5 | Estrutura das páginas |
| CSS3 | Estilização e animações |
| JavaScript | Funcionalidades interativas |
| Font Awesome 6 | Ícones vetoriais |
| Google Fonts | Tipografia (Great Vibes + Raleway) |
| LocalStorage | Persistência do carrinho |
| WhatsApp API | Comunicação com clientes |
